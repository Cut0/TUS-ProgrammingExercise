kadai-\[version\]をコンパイルして実行してください

# TIPS

### 命名規則

グローバル変数はパスカルケース

```
Head,PowDiみたいなの
```

ローカル変数はキャメルケース

```
lastBoardNumsみたいなの
```

関数はスネークケース

```
find_next_boardsみたいなの
```

### 注意点

- 変数のスコープはできるだけ小さくする。
- 関数の機能は最小限に抑える。複数の機能を持たせない。
- ネストは 3〜4 重くらいが限度。できるだけ浅くする。
- 全てを関数化すると機能を辿るのが煩わしくなるため、いい感じにする。
